# Blenderポートフォリオガチャ

## 概要
私が制作したBlender/3Dプリンター作品を閲覧できるガチャ

## 開発環境/技術選定理由
* HTML/CSS：Webサイトのフロントエンドで選択。
* JavaScript：主にガチャ演出に使用。
* Python：主にガチャの抽選処理に使用。
* Flask：バックエンドPythonで抽選を行うため選択。Djangoと比較したうえで、今回はログイン等の機能を作らないため、継続的な編集のしやすさを最優先してFlaskを選択。
* Blender：3Dモデリングに使用。モデルをレンダリングした画像をガチャ結果に表示させている。

## 機能
* 単発/10連選択ボタン：ガチャを単発で回すか10連で回すか選択。
* 抽選：Flask・Pythonで抽選処理を行い、JavaScriptでカードをめくる演出を制御している
* 結果一覧：カード画像をクリックすると全画面表示される。一部画像はカルーセル表示にも対応。単発/10連選択ボタンを押すことで、再度抽選を行うことができる。
